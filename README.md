# Organização do Código

A estrutura de pastas será organizada da seguinte forma:

/projeto
│
├── /img          → imagens do site  
├── /css          → arquivos de estilo (CSS)  
├── /javascript   → scripts JavaScript  
└── /bootstrap    → arquivos e dependências do Bootstrap

#Controle de Versão
